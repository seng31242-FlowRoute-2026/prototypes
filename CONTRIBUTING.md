# Contribution Guidelines

## Design Standards

- Keep UI minimal and professional
- Maintain consistent spacing and typography
- Prioritize usability over visual complexity

---

## File Naming

Examples:

- client-upload-wireframe.fig
- pod-dashboard-v1.png
- navigation-flow.drawio

---

## Export Rules

For every Figma screen:

- Export PNG or SVG
- Include editable source where possible

---

## Commit Examples

feat(ui): add client upload page wireframe

docs(prototype): update dashboard navigation flow

fix(ui): improve pod workload card layout
