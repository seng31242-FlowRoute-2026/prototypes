# FlowRoute – Prototypes Repository

This repository contains all UI/UX prototype assets for the FlowRoute project.

## Repository Purpose

This repository stores:

- Wireframes
- Low-fidelity prototypes
- Figma exports
- Navigation flows
- UI experiments
- Dashboard concepts

## Folder Structure

```
prototypes/
├── wireframes/
├── figma-exports/
├── navigation-flows/
├── dashboard-designs/
└── assets/
```


## Design Tools

- Figma
- draw.io
- Excalidraw

## Design Goals

The FlowRoute UI focuses on:

- Simplicity
- Fast task visibility
- Clear workflow monitoring
- Minimal client friction
- Real-time workload visibility
